## Introduction
This is a platformer game set in an airport.
You must run, dash across gaps, jump around obstacles, and try to catch your flight - it's leaving very soon!

Along the way, you'll need to press buttons, activate moving platforms, and retrieve your lost luggage.

## How to Play
Download the jar file here: [https://github.com/cometkaizo/CatchTheFlight/raw/refs/heads/master/out/artifacts/CatchTheFlight_jar/CatchTheFlight.jar](url)

Simply run the jar file to start playing!

Note that there is no save system.

This program requires Java 25.

## Controls
Running: WASD

Dashing: SPACE

Interact: E

Throwing: LEFT CLICK while you are holding your luggage, and it will be thrown _in the direction of your mouse_!
